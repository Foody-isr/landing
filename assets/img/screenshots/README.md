# Screenshots

Place your product screenshots here.

## Recommended Format
- Format: PNG or JPG
- Dimensions: 1200x800 or higher (maintain 3:2 aspect ratio)
- File naming: lowercase with hyphens (e.g., `pos-dashboard.png`, `qr-ordering-flow.jpg`)

## Examples
- `pos-dashboard.png` - Main POS dashboard
- `table-view.png` - Table management interface
- `order-creation.png` - Order creation flow
- `kitchen-display.png` - Kitchen display system
- `guest-qr-menu.png` - Guest-facing QR menu
- `order-tracking.png` - Real-time order tracking

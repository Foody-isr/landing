# Videos

Place your product demo videos here.

## Recommended Format
- Format: MP4 (H.264 codec)
- Dimensions: 1920x1080 (Full HD) or 1280x720 (HD)
- Duration: Keep under 2 minutes for web performance
- File size: Optimize for web (under 10MB if possible)
- File naming: lowercase with hyphens (e.g., `demo-overview.mp4`)

## Examples
- `demo-overview.mp4` - Complete product overview
- `qr-ordering-demo.mp4` - QR ordering workflow
- `pos-workflow.mp4` - POS operation demo
- `kitchen-flow.mp4` - Kitchen ticket workflow

## Notes
- Consider hosting larger videos on YouTube/Vimeo and embedding instead
- For GitHub Pages, keep file sizes reasonable
- Add a poster image (thumbnail) for each video
